## Installation

1. Clone the project to the folder you like

```
cd your_folder/
git clone https://github.com/vek/Graffitis.git

```
2. Go to project folder
```
cd Graffitis

```
3. Install packages
```
meteor npm install

```
4. Run meteor
```
meteor run

```
